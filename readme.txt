This simple reminder application will analyse user input and schedule a reminder based on the given values.

Getting Started:

1. Open simple-reminder\src\main\resources\prerequisites.sql and execute the query.
2. Open simple-reminder\src\main\java\quartz\QuartzProperties.java, configure your database credentials and execute the program.
3. Give the same configuration for ScheduleJob.java and execute the program with different inputs.

Note: This is a basic version. You can implement further with many options. Please give suggestions.
